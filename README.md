# Chase Game - Python Port

Портированная версия классической игры Chase с BASIC на Python с поддержкой множества платформ.

##Описание проекта

Это точный порт классической игры Chase, написанной на BASIC в 1970-х годах. Игра была портирована на Python с сохранением оригинальной логики и поведения.

### История
Игра Chase была опубликована в книге "BASIC Computer Games" (1978) под редакцией Дэвида Ала. Это одна из первых компьютерных игр, демонстрирующих простой искусственный интеллект.

### Особенности порта:
- **Точное соответствие** оригинальной BASIC версии
- **Автоматическое тестирование** с golden master сравнением
- **Кроссплатформенность** (Windows, Linux, Web, Android)
- **Поддержка расширенных платформ** (Plan9, boot sector, fairy CPU)

## Игровой процесс

Вы - символ `*` в лабиринте высокого напряжения. Пять перехватчиков (`+`) пытаются вас уничтожить. Стены лабиринта (`X`) смертельно опасны. Ваша цель - заманить перехватчиков на стены.

### Управление:
7 8 9 (вверх-влево, вверх, вверх-вправо)
4 * 6 (влево, остаться на месте, вправо)
1 2 3 (вниз-влево, вниз, вниз-вправо)

Специальные команды:
0 - Случайный прыжок
10 - Пропуск ходов до конца игры
-1 - Сдаться

## Установка и запуск

### Базовые требования:
- Python 3.7 или выше
- pip (менеджер пакетов Python)

### Установка зависимостей:
# Клонирование репозитория
git clone https://github.com/ваш-username/chase-game.git
cd chase-game

# Установка зависимостей
mingw32-make install

###Запустите веб-версию:
mingw32-make web-install
mingw32-make run-web

Откройте браузер и перейдите по адресу: http://localhost:5000

###Linux 
make run-linux

###Windows
mingw32-make run-windows
